# XT_Note

A note application for XT.

# 技术栈

- Vue3  
- Vite
- TypeScript
- Element Plus
- Pinia
- Axios

# 实现功能

- 暗夜模式  Element Plus
- 数据持久化 Pinia & LocalStorage
- 笔记列表 v-if
- 随机背景 axios
